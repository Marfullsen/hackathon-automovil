Descripción del Proyecto
2/9

Objetivo

Simulador de conducción interactivo en 3D diseñado para la educación vial. Permite a los usuarios practicar maniobras, reconocer señales de tránsito y entender las reglas de prioridad en un entorno seguro y virtual.

Visión

Proporcionar una plataforma educativa accesible que transforme la forma en que se enseña y aprende la seguridad vial a través de la tecnología 3D inmersiva.

Público Objetivo

  • Estudiantes de programas OTEC
  • Postulantes a Licencia Clase B
  • Usuarios de todas las edades interesados en mejorar sus habilidades de conducción
Basado en Three.js y JavaScript moderno
Características Principales
3/9
🌐
Entorno 3D Inmersivo
Basado en el motor de renderizado de Bruno Simon, optimizado para web con máximo rendimiento.
⚙️
Sistema de Físicas Realista
Manejo de colisiones, fricción de neumáticos y gravedad para simulación auténtica.
🏆
Módulo de Logros
Sistema integrado para trackear progreso (respeto de semáforos, velocidad permitida, etc.).
📱
Soporte Multidispositivo
Controles adaptados para teclado y dispositivos móviles con botones táctiles.
Tecnología WebGL optimizada para académicos
Instalación y Configuración
4/9

1. Clonar el Repositorio

git clone https://github.com/tu-usuario/vial-sim-game.git
cd vial-sim-game

2. Instalar Dependencias

npm install

3. Ejecutar en Modo Desarrollo

npm run dev

4. Construir para Producción

npm run build
Requiere Node.js y npm instalados

Estructura del Proyecto
6/9

Componentes Principales

  • index.html - Punto de entrada con contenedor canvas y UI
  • /assets - Scripts de lógica principal
  • /respawns - Modelos .glb comprimidos para objetos y vehículos
  • /ui - Iconos SVG y previsualizaciones para navegación

Características de Arquitectura

  • Basada en HTML5 para máxima compatibilidad
  • Motor 3D: Three.js
  • Optimización procedural de bloques urbanos
  • Soporte WebGL para renderizado acelerado
Arquitectura modular y escalable
Roadmap de Aprendizaje
7/9
📍 NIVEL 1: Fundamentos

Reconocimiento de señales de pare y ceda el paso. Aprendizaje de reglas básicas de tránsito urbano.

📍 NIVEL 2: Intermedio

Circulación en rotondas y prioridades complejas. Manejo de intersecciones con múltiples direcciones.

📍 NIVEL 3: Avanzado

Conducción nocturna y condiciones climáticas adversas. Implementación de texturas .ktx y iluminación dinámica.

✓ Escalabilidad futura: IA para generar escenarios basados en errores históricos del usuario

Aprendizaje progresivo y adaptativo
Innovación y Ventajas
8/9

✓ Fortalezas
